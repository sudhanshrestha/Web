Home page

The designing of the home page of my website was taken from crypton.trading as a refrence. The site had a automated typing effect with the letters that are being displayed with a black background which I implemented in my site as well where I displayed the automatic typing effect with my name and added a navigation bar right below the effect for navigation between the pages. As the refrence site had many features to not make it as just a plane old black dull site I too added a partical.js plugin in the home page so that it does not become just a plane old dull black page.
Navigation bar

The refrence I took from the site maxreeves.net was the navigation bar style and the page designs. The navigation and the page style of maxreeves.net was simple and minimalistic which i liked so i implemented it in my website most of my website design was done with the thought of keeping it simple. I added the navigation with some hover effect and instead of putting the branding of the site in the middle I kept it in the top left . I made the look of the navigation bar a bit big for the hover effect to look a bit better.

About page
